skbio.sequence.NucleotideSequence¶

class skbio.sequence.NucleotideSequence(sequence, id='', description='', quality=None, validate=True, case_insensitive=False)[source]¶

Abstract base class for storing an IUPAC nucleotide sequence.

This is an abstract base class (ABC) that cannot be instantiated.

See also

DNA, RNA

Attributes

description Biological sequence description.
sequence Array containing underlying biological sequence characters.

nondegenerate_chars Return non-degenerate IUPAC characters.
degenerate_chars Return degenerate IUPAC characters.
degenerate_map Return mapping of degenerate to non-degenerate characters.
complement_map Return mapping of nucleotide characters to their complements.
