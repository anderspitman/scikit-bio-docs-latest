skbio.sequence.IUPACSequence¶

class skbio.sequence.IUPACSequence(sequence, id='', description='', quality=None, validate=True, case_insensitive=False)[source]¶

Store biological sequence data conforming to the IUPAC character set.

This is an abstract base class (ABC) that cannot be instantiated.

Raises:

ValueError

If sequence characters are not in the IUPAC character set [R174].

See also

NucleotideSequence, DNA, RNA, Protein

References

[R174](1, 2) Nomenclature for incompletely specified bases in nucleic acid sequences: recommendations 1984. Nucleic Acids Res. May 10, 1985; 13(9): 3021-3030. A Cornish-Bowden
